## ðŸ”§ Local Setup

### 1. Clone the Repo
```bash
git clone https://github.com/Spectra010s/steel.git
cd steel
````

### 2. Run Backend (Socket.io)

```bash
cd backend
npm install
node index.js
```

### 3. Run Frontend (Next.js)

```bash
cd ../frontend
npm install
npm run dev
```

* Chat should be live on `http://localhost:3000`
* Backend runs on `http://localhost:5000` (default)
